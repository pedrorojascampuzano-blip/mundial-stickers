# Mi Álbum Mundial 2026

App para juntar las estampas del álbum Panini FIFA World Cup 2026 (980 estampas, 48 selecciones en 12 grupos + especiales).

- Marca las que ya tienes, lleva tus repes para cambios, ve tu progreso por equipo / grupo / total.
- Pestaña "Me faltan" y "Repes" con botón de copiar para mandar por WhatsApp.
- Funciona offline. El progreso se guarda en tu dispositivo. Export/import para pasarlo al celular.

Abrir: `index.html` o la versión publicada en GitHub Pages.
